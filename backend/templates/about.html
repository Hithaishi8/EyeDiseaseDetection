About EyeSightAI

A compact explanation of what the system does and how it's useful in clinics and screening camps.

Model summary

  • CNN + wavelet hybrid classifier for multi-disease detection
  • U-Net segmentation for lesion localization
  • Postprocessing: morphological filtering, bounding box extraction, confidence scoring

Deployment

Runs on CPU/GPU. Lightweight inference API compatible with Flask backends.

Outputs

  • Prediction label + confidence
  • Segmentation mask and bounding boxes
  • Clinical summary PDF
